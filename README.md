# cssmin-go

## Why?

Because I hate node.js

## How?

### Building

```bash
export GOPATH=/foo
go get github.com/dchest/cssmin
go build
```

## Using

```bash
cat file.css | cssmin-go
```

## Anything else?

Using [dchest/cssmin](https://github.com/dchest/cssmin). Thanks, dchest!
